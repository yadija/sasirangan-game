# Sasirangan Game

Ini adalah aplikasi web untuk mempelajari dan memainkan quiz yang berhubungan dengan sasirangan.

## Teknologi yang digunakan

- Typescript
- React
- Tailwind CSS
- Shadcn UI

## Fitur

- Memainkan quiz
- Mempelajari materi
- Melihat peringkat
- Tema gelap
- Audio

## Instalasi

1. Clone repository
2. Install dependencies `npm install`
3. Run `npm run dev`
4. Open `http://localhost:5173` in your browser
